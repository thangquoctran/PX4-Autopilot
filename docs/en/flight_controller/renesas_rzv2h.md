The Renesas RZ/V2H Evaluation Kit runs FreeRTOS on the Arm Cortex-R8 core.
PX4 can be built using the `arm-none-eabi` cross compiler toolchain.

## Quick Summary
- CPU: Arm Cortex-R8 (32-bit)
- Runs FreeRTOS
- Build target: `renesas_rzv2h_default`
- Toolchain: `arm-none-eabi`
